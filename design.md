# Portfolio Design Prompt

Use this as the design brief for this portfolio.

## Overall Style
- Editorial, cinematic, and slightly experimental.
- Dark-first visual system with a polished terminal/monitor feel.
- Minimal, premium, and content-led rather than decorative-heavy.
- Strong contrast, generous whitespace, and precise spacing.
- Subtle futuristic accents without looking neon or cyberpunk.

## Visual Direction
- Base background: near-black with very soft depth.
- Primary text: off-white, high contrast.
- Secondary text: muted gray for supporting copy and labels.
- Accent color: electric blue / indigo for highlights, nav numbers, status dots, and small interface details.
- Success/status accent: soft green for availability indicators.
- Surfaces: translucent dark panels, faint borders, and restrained glow.
- Add a delicate grid overlay and subtle noise texture to avoid flat backgrounds.

## Typography
- Primary sans: Geist-style clean UI sans for body and interface text.
- Display serif: Instrument Serif-style elegant serif for large headings.
- Mono: JetBrains Mono-style font for navigation, labels, metadata, and technical UI.
- Hero heading should feel large, sculptural, and editorial.
- Section headings should use serif display styling with tight tracking and strong hierarchy.
- Supporting text should stay crisp and readable, with moderate line height.

## Layout System
- Fixed top navigation with compact branding, section links, and theme control.
- Full-viewport landing section with hero content and ticker stacked vertically.
- Sections separated by thin rules and large vertical spacing.
- Content width should stay centered in a max-width container.
- Use a clean 12-column feeling for major content blocks, but keep the implementation simple and responsive.
- Preserve strong left alignment and structured rhythm.

## Hero Treatment
- Hero should dominate the first viewport.
- Large serif name split over two lines on desktop and still readable on smaller screens.
- Include a short role line and a longer supporting description.
- Add a small status indicator for availability.
- Include two clear CTA buttons: primary filled and secondary outlined.
- Put a ticker/marquee directly under the hero inside the initial viewport.
- Use subtle animated background glows and a grid texture behind the hero.

## Section Patterns
- Each section begins with a small eyebrow label and number.
- Section titles should feel editorial and intentional.
- Use thin borders, subtle dividers, and minimal card chrome.
- Keep section layouts content-first and highly scannable.

## Components And Surfaces
- Navigation: fixed, minimal, numbered section links, scroll progress bar, corner markers.
- Theme picker: compact swatch-based control with a floating panel or mobile FAB.
- About: two-column layout with fact list on the left and bio/statistics on the right.
- Skills: category pills and a grid of skill tiles with gentle hover states.
- Projects: list-style rows with a modal detail view on select.
- Experience: timeline-like rows with period, role, company, and bullet highlights.
- Contact: social links with hover scramble effect plus a clean form with bottom borders.
- Footer: slim, understated, and separated by a top rule.

## Motion And Interaction
- Use Framer Motion-style fade, stagger, and slide-in animations.
- Hero name can use a scramble/reveal effect on load or hover.
- Ticker should move continuously and smoothly.
- Hover states should be subtle: slight lift, border change, color shift, or underline.
- Use smooth scrolling for section navigation.
- Keep motion elegant and restrained; avoid bouncy or playful animation.

## Background And Texture
- Main background should stay nearly black.
- Add a grid pattern at low opacity.
- Add layered radial glows with slow breathing animation.
- Add a subtle paper/noise texture overlay.
- Keep all texture effects low-contrast so they support the content instead of competing with it.

## Design Keywords

- cinematic
- dark minimal
- terminal-inspired
- serif + mono contrast
- subtle glow
- high contrast
- structured spacing
- premium portfolio
- modern personal site

## Shareable Prompt Version
Design a dark editorial portfolio website with a premium terminal-inspired look. Use a near-black background, off-white text, electric blue accents, subtle green status indicators, and a mix of clean sans-serif, elegant serif display typography, and monospace metadata text. The layout should feel full-screen and spacious, with a fixed numbered navigation bar, a dramatic hero section, a continuous ticker, thin divider lines, and content-led sections for About, Skills, Projects, Experience, and Contact. Add soft glows, a faint grid texture, and noise for depth. Keep interactions restrained and polished with smooth scrolling, subtle hover states, staggered reveals, and minimal but intentional motion.
